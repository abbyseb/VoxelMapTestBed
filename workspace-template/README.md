# VoxelMap TestBed — sample workspace

Copy this layout into your project root (or run **TestBed: Initialize Workspace** from the command palette).

```
your-project/
├── models/
│   ├── baselines/     # read-only reference architectures (Phase 3+)
│   └── custom/        # your editable model code
├── modules/           # shared blocks (FiLM, attention, …)
├── experiments/       # YAML experiment specs
└── runs/              # train/test outputs (created by TestBed)
```

Open this folder in VSCodium as the workspace when using the TestBed extension.
